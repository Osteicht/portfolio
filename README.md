### 🚀 What I Do
I build functional, data-driven web applications with a strong focus on **backend performance and reliability rather than complex designs**. I am a practical developer—my goal is to create decent, working websites that handle user data safely and run efficiently under the hood.

My standout project is **Stats Poll Awards**, a real-time voting platform built to handle active traffic. If you need a solid foundation, secure database routing, and a website that simply does its job well, that is my strong suit.

### 🛠️ My Tech Stack & Experience
Web development is broad, so here are the specific tools and frameworks I use to build my applications:

*   **Backend / Server:** Node.js, Express.js
*   **Database:** PostgreSQL 
*   **Frontend Interface:** HTML, CSS, and EJS (Templating Engine)
*   **Real-Time Data:** Socket.io (for live voting/updates)
*   **Web Security:** I implement standard app protections including `helmet` (HTTP headers), `cors`, `express-validator` (input sanitization), `rate-limiter-flexible` (brute-force protection), and Google OAuth2 (`passport`) for secure user logins.
*   **Deployment Platforms:** Render (for full-stack apps) and GitHub Pages (for static sites)
